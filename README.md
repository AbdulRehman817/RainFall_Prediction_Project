# 🌧️ Rainfall Prediction using Machine Learning

A Machine Learning web application that predicts whether rainfall is likely to occur based on weather conditions. The project includes data preprocessing, model training, and deployment using Streamlit.

## 🚀 Live Demo

https://abdulrehman817-rainfall-prediction-project-app-opgiwm.streamlit.app

---

## 📌 Features

- Predicts rainfall using a trained Machine Learning model
- Interactive Streamlit web interface
- Real-time predictions
- Clean and user-friendly UI
- Preprocessing with saved scaler and feature columns
- Model deployed using Streamlit Cloud

---

## 🛠️ Technologies Used

- Python
- Pandas
- NumPy
- Scikit-learn
- Joblib
- Streamlit

## ⚙️ Installation

Clone the repository

```bash
git clone https://github.com/AbdulRehman817/RainFall_Prediction_Project.git
```

Move into the project folder

```bash
cd RainFall_Prediction_Project
```

Install the required libraries

```bash
pip install -r requirements.txt
```

Run the Streamlit application

```bash
streamlit run app.py
```

---

## 📊 Workflow

1. Load the trained model.
2. Load the scaler and feature columns.
3. Accept user input from the Streamlit interface.
4. Scale the input data.
5. Predict rainfall using the trained model.
6. Display the prediction to the user.

## 📈 Machine Learning Pipeline
